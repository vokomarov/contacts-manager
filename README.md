# Contacts Manager

## Requirements

- PHP 7.3 || 8.0
- Docker and Docker Compose

## Install

- `cp .env.example .env` and verify settings
- `composer install`
- `sail up`
- `sail composer install`
- `sail npm run dev`
- `sail php artisan migrate`
- `sail php artisan db:seed`

## Tests

- `sail php artisan test`
